# Invoice Management System

A web-based invoice management system built with PHP and MariaDB.

## Features

- Multi-company support with role-based access
- User authentication and authorization
- Customer and item management
- Invoice generation and tracking
- Government portal API integration
- Admin and Accountant roles

## Setup

1. Import the database schema from `database/schema.sql`
2. Configure database connection in `config/database.php`
3. Set up your web server to point to the project root
4. Access the application via your web browser

## Default Login
- Admin: admin@company.com / admin123
- Accountant: accountant@company.com / acc123
